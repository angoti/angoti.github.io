O que você vai fazer

Preparar o ambiente de desenvolvimento de aplicações React Native em um sistema operacional Windows 10

Referências

  • Node.js
  • Expo
  • Java SE Development Kit 17 latest JDK LTS
  • Visual Studio
  • Android Emulator

Vamos usar a versão LTS (long-term support) mais recente que é 16.17.0. Baixe o instalador direto do site site oficial. Após a instalação, você poderá verificar a instalação com o comando:

node -v

Depois de instalar o node, precisamos instalar Expo CLI: 

npm install --global expo-cli

Para executar a aplicação no celular é necessário instalar o app Expo na Google Play ou na App Store.

O Java 17 LTS (long-term support) é a versão LTS mais recente para a plataforma Java SE. O JDK 17 receberá atualizações até pelo menos setembro de 2024. Baixe o instalador e instale o JDK para rodar o emulador Android.

Baixe o instalador do vscode e depois instale as seguintes extensões:

Prettier - Code formatter

IntelliCode

JavaScript (ES6) code snippets

Babel JavaScript

Node.js Modules Intellisense

ESLint

SonarLint
