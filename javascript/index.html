Referência

JavaScript é uma linguagem de programação essencial no desenvolvimento web moderno. Criada inicialmente para adicionar interatividade a páginas HTML, ela evoluiu para se tornar uma linguagem poderosa, capaz de atuar tanto no frontend (lado do cliente) quanto no backend (lado do servidor, com Node.js).

No contexto da web, o JavaScript permite que desenvolvedores manipulem a árvore DOM (Document Object Model), respondam a eventos do usuário (como cliques e digitação), e criem interfaces interativas sem a necessidade de recarregar a página. É a linguagem responsável por transformar páginas estáticas em aplicações interativas e responsivas.

Por que aprender JavaScript?

  • Padrão da indústria: todos os navegadores modernos suportam JavaScript nativamente;
  • Versatilidade: com o ecossistema atual, é possível desenvolver aplicações completas apenas com JavaScript (Full Stack);
  • Integração com APIs: é a linguagem ideal para consumir e manipular dados de serviços externos;
  • Base para frameworks modernos: bibliotecas como React, Vue e Angular são escritas em JavaScript e amplamente utilizadas no mercado.

JavaScript é baseado na especificação ECMAScript, que padroniza sua sintaxe e funcionalidades. Isso garante consistência e interoperabilidade entre diferentes implementações da linguagem.

Conceitos-chave iniciais

  • Variáveis e tipos de dados: let, const, strings, números, booleanos.
  • Controle de fluxo: if, else, switch, laços como for e while.
  • Funções e escopo: função tradicional e arrow functions (=>).
  • Manipulação do DOM: acessar e alterar elementos HTML via document.getElementById, querySelector, etc.
  • Eventos: capturar interações do usuário com addEventListener.
  • Programação assíncrona: uso de setTimeout, fetch, async/await.

Exemplo prático: Interatividade com JavaScript

HTML

<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <meta charset="UTF-8">
  <title>Exemplo JS Interativo</title>
</head>
<body>
  <h1 id="titulo">Olá, visitante!</h1>
  <button id="btn">Clique aqui</button>

  <script src="script.js"></script>
</body>
</html>

JavaScript (script.js)

const titulo = document.getElementById("titulo");
const botao = document.getElementById("btn");

botao.addEventListener("click", () => {
  titulo.textContent = "Você clicou no botão!";
});

Nesse exemplo, o JavaScript escuta o evento de clique no botão e altera dinamicamente o texto do título. Este é o primeiro passo para criar aplicações mais ricas, como jogos, simuladores, dashboards ou formulários inteligentes.

Declaração de variáveis

  • let: usada para declarar variáveis que podem ter seu valor alterado. Possui escopo de bloco.
  • const: usada para declarar constantes, ou seja, variáveis cujo valor não pode ser reatribuído. Também possui escopo de bloco.
  • var: forma mais antiga de declarar variáveis. Possui escopo de função e não respeita blocos. Pode causar comportamentos inesperados e deve ser evitada em código moderno.
let nome = "Ana";
const idade = 20;
var cidade = "São Paulo";

Escopo em JavaScript

Escopo é o local onde uma variável é válida e acessível no código. Existem três principais tipos:

  1. Escopo Global: fora de qualquer função ou bloco.
  • A variável pode ser acessada de qualquer lugar no código.
  1. Escopo de Função: variáveis declaradas dentro de uma função com var, let ou const só existem dentro daquela função.
  2. Escopo de Bloco: variáveis declaradas com let ou const dentro de {} (como if, for, etc.) só existem ali dentro.
let nome = "Ana"; // escopo global

function saudacao() {
  let mensagem = "Olá"; // escopo de função
  if (true) {
    const linguagem = "JavaScript"; // escopo de bloco
    console.log(mensagem + " " + nome + " que usa " + linguagem);
  }
  // console.log(linguagem); // Erro: linguagem não está acessível aqui
}

saudacao();

Objetivos

  • Utilizar estruturas condicionais para controlar o fluxo do programa
  • Compreender o uso de if, else if, else e switch

1. Condicional if

let idade = 18;

if (idade >= 18) {

console.log("Maior de idade");

}

2. Condicional if...else

let hora = 10;

if (hora < 12) {

console.log("Bom dia!");

} else {

console.log("Boa tarde!");

}

3. Condicional if...else if...else

let nota = 7;

if (nota >= 9) {

console.log("Excelente");

} else if (nota >= 7) {

console.log("Bom");

} else {

console.log("Precisa melhorar");

}

4. Estrutura switch

let dia = 3;

switch (dia) {

case 1:

console.log("Domingo");

break;

case 2:

console.log("Segunda-feira");

break;

case 3:

console.log("Terça-feira");

break;

default:

console.log("Outro dia");

}
