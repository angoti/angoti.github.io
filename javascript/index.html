Conseguimos acessar os elementos em HTML se dermos nomes a eles. E para acessar esses elementos precisamos então usar a linha de comando: document.getElementById("<nome do elemento>")

Não é necessário a instalação ou configuração de nenhuma biblioteca ou programa para a utilização e/ou interpretação de scripts no brower. Por default os browser interpretam linguagens em JavaScript.

O bloco de <script> pode ser adicionado ao elemento <head> e/ou <body> porém deve-se tomar cuidado quando utilizar ele dentro do elemento body. Esses cuidados serão apresentados nos próximos tópicos.

Comentários em javascript

Comentários são trechos de texto inseridos no código-fonte que não são interpretados pelo compilador ou interpretador, sendo destinados a fornecer explicações e informações adicionais para os programadores que irão ler e manter o código no futuro.

No JavaScript existem dois tipos de comentários: linha e em bloco.

Linha:

// Comentário

Bloco:

/*

Comentário de bloco

Ai não precisa ficar colocando o

// em cada linha.

*/

Caixas de diálogo

Em Javascript pode-se utilizar as caixas de diálogos padrões para que tenham uma interação pequena com o usuário.

<!DOCTYPE html>
<html>
        <body>
                <h1>My First Web Page</h1>
                <p>My first paragraph.</p>
                <script>
                        window.alert(‘alô mundo');
                </script>
        </body>
</html>

Caixa utilizada somente para apresentar uma mensagem para o usuário na tela,, mas nesse momento o usuário poderá responder sim ou não. Como por exemplo: Desejas continuar? 

<!DOCTYPE html>
<html>
        <body>
        <p>Clique no botão: </p>

        <button onclick="enviar()">Enviar</button>
        <p id="demo"></p>

        <script>
                function enviar() {
                  let text;
                  if (confirm("Deseja continuar") == true) {
                        text = "OK!";
                  } else {
                        text = "Não OK";
                  }
                  document.getElementById("demo").innerHTML = text;
                }
        </script>
        </body>
</html>

Caixa utilizada permite a entrada de dados do usuário que poderá ser usada em outro momento no programa. 

<!DOCTYPE html>
<html>
        <body>
        <p>Clique no botão:</p>
        <button onclick="bebida()">Clique aqui</button>
        <p id="demo"></p>

        <script>
                function bebida() {
                  let text;
                  let favorita = prompt("Qual a sua bebida favorita", "Coca-Cola");
                  switch(favorita) {
                        case "Coca-Cola":
                          text = "Você escolheu coca.";
                          break;
                        case "Pepsi":
                          text = "Você escolheu Pepsi";
                          break;
                        case "Sprite":
                          text = "Você escolheu Sprite";
                          break;
                        default:
                          text = "Você escolheu outra bebida";
                  }
                  document.getElementById("demo").innerHTML = text;
                }
        </script>
        </body>
</html>

Variáveis são contêineres para armazenar valores. Em Javascript temos duas características simplórias das utilizações das variáveis:

  • Tipagem fraca: Não há declaração antecipada do tipo de dados.
  • Tipagem dinâmica: variável Javascript pode armazenar diferentes tipos de dados em diferentes momentos dependendo do contexto do problema.

Variáveis em JavaScript podem ser declaradas de 4 maneiras:

  • Automaticamente
  • Usando var
  • Usando let
  • Usando const

Automaticamente

Neste primeiro exemplo, x, y e z são variáveis não declaradas. Eles são declarados automaticamente quando usados pela primeira vez:

<script>
x = 5;
y = 6;
z = x + y;
window.alert("o valor de z eh: " + z);
</script>
<script>
var x = 5;
var y = 6;
var z = x + y;
window.alert("o valor de z eh: " + z);
</script>

Usando let e const

Deve-se sempre declarar variáveis. Sempre use const se o valor não deve ser alterado ou se o tipo não deve ser alterado (Arrays e Objetos). E sempre use let quando não puder usar const.

As duas variáveis price1 e price2 são declaradas com a palavra-chave const. Esses são valores constantes e não podem ser alterados. A variável total é declarada com a palavra-chave let. O valor total pode ser alterado.

<script>
const price1 = 5;
const price2 = 6;
let total = price1 + price2;
window.alert (o total eh: " + total);
</script>

Tipos primitivos de dados

O Javascript possui somente três tipos primitivos de dados: string, numeric e boolean. Portanto possui os mesmos objetos internos: String, Numeric e Boolean.

O objeto String envolve o tipo primitivo string, assim como os objetos Number e Boolean o fazem com seus tipos de dados primitivos individuais, numeric e boolean, respectivamente.

Para ficar mais claro vamos ao exemplo:

  1. var nome = "carlos";
  2. var nomeMaiusculas = nome.toUpperCase();

No trecho de código acima, quando o método toUpperCase é chamado por "nome.", um objeto é criado para envolver a string e a seguir processar a chamada do método antes que o objeto seja descartado. Para todos os propósitos, nome se parece com um objeto e atua como um objeto ao chamar o método, mas é um tipo primitivo.

Referência
