Objetivos da Aula

  • Compreender o que é o Bootstrap e sua importância no desenvolvimento web.
  • Construir páginas responsivas usando o sistema de grid e componentes.
  • Conhecer outros frameworks CSS e comparar suas características.

Introdução ao Bootstrap

  • Framework CSS open source criado pelo Twitter.
  • Facilita a criação de interfaces modernas e responsivas.
  • Trabalha em conjunto com HTML, CSS e JS.

Por que usar o Bootstrap?

  • Acelera o desenvolvimento: Com uma vasta coleção de componentes e estilos prontos, você não precisa programar tudo do zero.
  • Responsividade: O sistema de grid (grade) do Bootstrap permite criar layouts que se ajustam automaticamente ao dispositivo do usuário.
  • Consistência: Garante uma aparência padronizada e profissional para seu site, mesmo sem ser um especialista em design.
  • Fácil de usar: É baseado em HTML, CSS e JS, exigindo apenas um conhecimento básico dessas tecnologias para começar.

O Bootstrap oferece uma ampla gama de componentes prontos para uso que aceleram o desenvolvimento:

  • Botões: diversos estilos e cores, ex.:
<button class="btn btn-primary">Salvar</button>
<button class="btn btn-outline-secondary">Cancelar</button>
  • Cards: usados para exibir informações agrupadas (como produtos, artigos ou perfis).
<div class="card" style="width: 18rem;">
  <img src="imagem.jpg" class="card-img-top" alt="...">
  <div class="card-body">
    <h5 class="card-title">Título do Card</h5>

    <a href="#" class="btn btn-primary">Detalhes</a>

</div>

<nav class="navbar navbar-expand-sm navbar-light bg-light">

Apps estilo Material Design

O sistema de grid do Bootstrap é baseado em 12 colunas e foi pensado para mobile‑first (você define o comportamento em telas pequenas e vai "abrindo" o layout conforme os breakpoints).

Conceitos‑chave

  • Container: área máxima do conteúdo. Use .container (largura máxima por breakpoint), .container-fluid (100% da largura) ou .container-{breakpoint} (fluido até certo ponto).
  • Row: agrupa colunas na horizontal e aplica gutters (espaços) automaticamente.
  • Column: define a largura do conteúdo. Pode ser automática (.col) ou explícita (.col-6, .col-md-4, etc.).

Exemplos

1) Container + linha + colunas (base)

<div class="container">
  <div class="row">
    <div class="col">A</div>
    <div class="col">B</div>
    <div class="col">C</div>
  </div>
</div>

.col divide o espaço restante igualmente entre as colunas.


2) Larguras fixas e responsivas

<div class="container">
  <div class="row">
    <div class="col-12 col-md-8">Conteúdo principal</div>
    <div class="col-12 col-md-4">Barra lateral</div>
  </div>
</div>

Em telas pequenas: stack (12/12). A partir de md: 8/12 + 4/12.

3) Gutters (espaçamentos entre colunas)

Use g-* (ambos eixos), gx-* (horizontal) e gy-* (vertical).
Valores vão de 0 a 5.

<div class="container">
  <div class="row g-3">
    <div class="col-6 col-lg-3"><div class="p-3 bg-light">1</div></div>
    <div class="col-6 col-lg-3"><div class="p-3 bg-light">2</div></div>
    <div class="col-6 col-lg-3"><div class="p-3 bg-light">3</div></div>
    <div class="col-6 col-lg-3"><div class="p-3 bg-light">4</div></div>
  </div>
</div>

4) Alinhamento vertical e horizontal

Na row: align-items-* (vertical), justify-content-* (horizontal). Em cada col: align-self-*.

<div class="container">
  <div class="row align-items-center justify-content-between" style="min-height: 160px;">
    <div class="col-4"><div class="bg-light p-3">Esquerda</div></div>
    <div class="col-4"><div class="bg-light p-3">Centro</div></div>
  </div>
</div>

5) Ordenação e offsets

Reordene visualmente com order-* e crie recuos com offset-*.

<div class="container">
  <div class="row">
    <div class="col-12 col-md-4 order-md-2">Coluna 2 (aparece depois em md+)</div>
    <div class="col-12 col-md-4 order-md-1">Coluna 1</div>
    <div class="col-12 col-md-4 offset-md-0">Coluna 3</div>
  </div>
</div>

Dica: order-0, order-1, order-2... e variantes por breakpoint (order-md-1).
Para recuos: offset-md-2 desloca a coluna em 2 colunas à direita no md.

6) Colunas automáticas e com conteúdo

.col-auto faz a coluna "abraçar" o conteúdo, enquanto as demais ocupam o restante.

<div class="container">
  <div class="row">
    <div class="col-auto"><div class="bg-light p-2">Label:</div></div>
    <div class="col"><input class="form-control" placeholder="campo amplo"></div>
  </div>
</div>

7) Aninhamento (rows dentro de colunas)

<div class="container">
  <div class="row">
    <div class="col-12 col-lg-8">
      <div class="row g-2">
        <div class="col-6"><div class="p-2 bg-light">A1</div></div>
        <div class="col-6"><div class="p-2 bg-light">A2</div></div>
      </div>
    </div>
    <div class="col-12 col-lg-4"><div class="p-2 bg-light">B</div></div>
  </div>
</div>

8) Containers: fixo vs fluido

  • Fixo (.container): largura máxima muda por breakpoint (legível em telas grandes).
  • Fluido (.container-fluid): sempre 100% (ótimo para apps que ocupam toda a largura).
<div class="container-fluid">
  <div class="row g-4">
    <div class="col-12 col-md-6 col-xxl-3"><div class="p-3 bg-light">Card 1</div></div>
    <div class="col-12 col-md-6 col-xxl-3"><div class="p-3 bg-light">Card 2</div></div>
    <div class="col-12 col-md-6 col-xxl-3"><div class="p-3 bg-light">Card 3</div></div>
    <div class="col-12 col-md-6 col-xxl-3"><div class="p-3 bg-light">Card 4</div></div>
  </div>
</div>

9) Padrões de layout comuns

Layout com sidebar dupla (mobile first):

<div class="container">
  <div class="row g-4">
    <aside class="col-12 col-lg-3">Esquerda</aside>
    <main  class="col-12 col-lg-6">Conteúdo</main>
    <aside class="col-12 col-lg-3">Direita</aside>
  </div>
</div>

Grade de produtos/cards responsiva:

<div class="container">
  <div class="row gy-4 gx-3">
    <div class="col-6 col-md-4 col-lg-3">Card</div>
    <div class="col-6 col-md-4 col-lg-3">Card</div>
    <div class="col-6 col-md-4 col-lg-3">Card</div>
    <div class="col-6 col-md-4 col-lg-3">Card</div>
  </div>
</div>

Boas práticas

  • Pense primeiro no mobile (comece com .col-12 e vá refinando nos breakpoints).
  • Prefira utilitários (g-*, p-*, m-*, text-*, d-flex) antes de criar CSS próprio.
  • Use aninhamento para composições complexas e gx/gy para controlar espaços independentes.
  • Evite linhas com soma de colunas > 12 (quebra de linha inesperada).
  • Teste em tamanhos reais (DevTools → Responsive Design Mode).

    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
    <script>
      // Estado simples do carrinho
      const cart = [];

      function atualizarBadge() {
        document.getElementById('badgeQtd').textContent = cart.length;
      }

      function formatBRL(n) {
        return n.toLocaleString('pt-BR', { style: 'currency', currency: 'BRL' });
      }

      function atualizarCarrinhoUI() {
        const lista = document.getElementById('listaCarrinho');
        if (cart.length === 0) {
          lista.innerHTML = '<p class="text-body-secondary">Seu carrinho está vazio.</p>';
          document.getElementById('totalCarrinho').textContent = 'R$ 0,00';
          return;
        }
        let total = 0;
        lista.innerHTML = '<ul class="list-group mb-3" id="ulCarrinho"></ul>';
        const ul = document.getElementById('ulCarrinho');
        cart.forEach((item, idx) => {
          total += item.price;
          const li = document.createElement('li');
          li.className = 'list-group-item d-flex justify-content-between align-items-center';
          li.innerHTML = `<span>${item.name}</span><span class="fw-semibold">${formatBRL(item.price)}</span>`;
          ul.appendChild(li);
        });
        document.getElementById('totalCarrinho').textContent = formatBRL(total);
      }

      function showToast() {
        const toastEl = document.getElementById('toastOk');
        const toast = new bootstrap.Toast(toastEl, { delay: 1500 });
        toast.show();
      }

      // Adicionar ao carrinho
      document.querySelectorAll('.btn-add').forEach(btn => {
        btn.addEventListener('click', () => {
          const name = btn.getAttribute('data-product');
          const price = parseFloat(btn.getAttribute('data-price'));
          cart.push({ name, price });
          atualizarBadge();
          atualizarCarrinhoUI();
          showToast();
        });
      });

      // Preencher modal de detalhes dinamicamente
      const modalDetalhes = document.getElementById('modalDetalhes');
      modalDetalhes.addEventListener('show.bs.modal', event => {
        const button = event.relatedTarget;
        const name = button.getAttribute('data-product');
        const price = parseFloat(button.getAttribute('data-price'));
        document.getElementById('tituloModal').textContent = name;
        document.getElementById('descricaoModal').textContent = `O ${name} é excelente para demonstrar componentes do Bootstrap.`;
        document.getElementById('precoModal').textContent = price.toFixed(2).replace('.', ',');
      });

      // Offcanvas: atualizar UI ao abrir
      const carrinho = document.getElementById('carrinhoOffcanvas');
      carrinho.addEventListener('show.bs.offcanvas', atualizarCarrinhoUI);
    </script>
